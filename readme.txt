This plugin uses the blocking service - ["Anti Ya. Advisor"](https://antisovet.ru). The work and purpose of the plugin is as follows - it helps to register the site on the Anti Ya. Advisor service and automatically installs the blocking code on the site.


== Description ==

Anti Ya.Advisor is a service that blocks the actions of the Yandex.Advisor extension, which shows competitors to your website visitors.

The code blocks the following page elements:

* [top bar](/assets/screenshot-3.png)
* [pop-up window with a list of competitors](/assets/screenshot-3.png)
* push notification

All the rules and regulations of the Anti Ya. Adviser blocking service can be studied [by the link](https://antisovet.ru/policy/).

== Benefits ==

* Does not harm SEO promotion
* Increases the conversion of visitors on the site
* Completely secure code
* The most favorable conditions
* Works in all popular browsers

== Installation ==

To get the blocker code, after installing and activating the plugin, you need to specify your E-mail on the settings page to which the blocking will be linked (by default, this is the E-mail specified in the site settings). After clicking on the "Set Code" button, a blocking code will be generated and automatically posted on the site.

The service blocking code can also be installed without using this plug-in, manual installation instructions [here](https://antisovet.ru/blog/blokirovka-yandeks-sovetnika-na-wordpress/). In fact, this plugin simplifies and automates the registration of the site and the insertion of the blocking code.

If you have any questions or problems, please contact the technical support of the module developer.

== Frequently Asked Questions ==

= What is Yandex.Advisor and what exactly is blocking our service? =

["Yandex.Advisor"](https://sovetnik.yandex.ru/welcome/) is an extension for browsers, for example for [Google Chrome](https://chrome.google.com/webstore/detail/советник-яндексмаркета/ppiaojpbclpegkkkmikabinlpbahhbha?hl=ru), which is based on the basis of Yandex.Market products and stores, by subject matter and prices. This extension can be installed in any browser of any system, thus, technically, each visitor, while browsing your site, will see more favorable offers from your competitors and can easily go to any other site, as a result of which you will lose a visitor. An example of [what the site looks like](/assets/screenshot-3.png) with the Yandex extension installed.

The blocking service is just struggling with the actions of the extension on the site. You can read more about Yandex.Advisor and the method of dealing with it [here](https://antisovet.ru/blog/chto-takoe-yandekssovetnik/).

= Does the lock code generate a load? =
No, the code is optimized and configured to be cached in the browser

= Does blocking affect site functionality or promotion =
No, the code works dynamically, and is aimed only at blocking Yandex.Advisor, so it does not participate in the functionality of the site

= Code does not block Yandex.Advisor =
The blocker is activated within 24 hours from the moment of installation on the site. This is done so that individual parameters can be applied for a specific site. In any case, if you have any questions or problems, you can always contact the plugin developer support service for help and advice.
